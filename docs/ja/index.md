# ラボ PC 環境セットアップハンドブック — 日本語

## はじめに

1. 下のリンクから使用している OS に合ったガイドを選んでください。
2. 順番に沿ってセットアップを進めてください。
3. 問題が発生した場合は [トラブルシューティング](setup/troubleshooting.md) ページを確認してください。
