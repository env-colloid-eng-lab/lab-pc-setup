# Setup Overview

This page gives you a recommended order for completing the environment setup.

## Recommended Setup Order

1. **Choose your OS guide** — [Windows / WSL](windows-wsl.md) or [macOS](mac.md)
2. **Install Miniforge** — [Miniforge](miniforge.md)
3. **Install VS Code and Python extension** — [VS Code & Python](vscode-python.md)
4. **Set up Git and GitHub** — [Git & GitHub](git-github.md)
5. **Enable GitHub Copilot** — [GitHub Copilot (Student)](copilot-student.md)

## Before You Start

- Make sure your operating system is up to date.
- You will need an internet connection throughout the setup.
- If you encounter any issues, check the [Troubleshooting](troubleshooting.md) page.
